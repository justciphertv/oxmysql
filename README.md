# oxmysql

> [!WARNING]
> This script may be unstable and contain bugs. We recommend using [CommunityOx/oxmysql](https://github.com/CommunityOx/oxmysql) on a live server

> [!NOTE]
> You need to add `add_unsafe_worker_permission oxmysql` to your server.cfg to allow the script to use worker threads

A fork of oxmysql using [mariadb](https://www.npmjs.com/package/mariadb) and [worker threads](https://nodejs.org/api/worker_threads.html) for improved performance.
